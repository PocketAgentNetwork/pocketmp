# Web Search Skill

## What This Does

This skill teaches you how to search the web effectively, analyze results, and properly cite sources. Essential for research, fact-checking, and staying informed.

## How to Use

### Basic Search Process

1. **Identify the Query**
   - What exactly are you looking for?
   - Use specific keywords
   - Consider synonyms and related terms

2. **Execute the Search**
   - Use available web search tools
   - Try multiple search queries if needed
   - Look for recent, authoritative sources

3. **Analyze Results**
   - Read multiple sources
   - Cross-reference information
   - Check publication dates
   - Verify author credibility

4. **Summarize Findings**
   - Extract key information
   - Synthesize from multiple sources
   - Present clearly and concisely

5. **Cite Sources**
   - Always include source URLs
   - Note publication dates
   - Credit authors when known

## Examples

### Example 1: Research Query

**User:** "What are the latest developments in AI agents?"

**Process:**
1. Search: "AI agents 2026 developments"
2. Find 3-5 recent articles
3. Read and analyze
4. Summarize key trends
5. Cite all sources

**Output:**
```
Recent AI agent developments include:

1. Multi-agent coordination systems (Source: TechCrunch, Feb 2026)
2. Improved reasoning capabilities (Source: ArXiv, Jan 2026)
3. Better tool use and integration (Source: OpenAI Blog, Feb 2026)

Sources:
- https://techcrunch.com/...
- https://arxiv.org/...
- https://openai.com/blog/...
```

### Example 2: Fact-Checking

**User:** "Is this claim true: [claim]?"

**Process:**
1. Search for the claim
2. Find authoritative sources
3. Check multiple perspectives
4. Verify with fact-checkers
5. Provide verdict with evidence

### Example 3: Comparison Research

**User:** "Compare React vs Vue for web development"

**Process:**
1. Search for recent comparisons
2. Look at official documentation
3. Check community sentiment
4. Analyze use cases
5. Present balanced comparison with sources

## Best Practices

### Search Strategy
- Start broad, then narrow down
- Use quotes for exact phrases
- Try different search engines
- Look beyond the first page
- Check publication dates

### Source Evaluation
- Prefer official sources
- Check author credentials
- Look for peer review
- Verify with multiple sources
- Note any biases

### Citation Format
- Include full URL
- Note publication date
- Credit author/organization
- Use inline citations
- List all sources at end

## Common Pitfalls

❌ **Don't:**
- Rely on a single source
- Ignore publication dates
- Skip source verification
- Forget to cite sources
- Present opinion as fact

✅ **Do:**
- Cross-reference information
- Check multiple sources
- Verify credibility
- Always cite sources
- Distinguish fact from opinion

## Advanced Techniques

### Deep Research
- Use academic databases
- Check primary sources
- Look for meta-analyses
- Review methodology
- Consider context

### Trend Analysis
- Compare multiple time periods
- Look for patterns
- Check historical data
- Note emerging themes
- Track changes over time

### Expert Finding
- Identify thought leaders
- Check credentials
- Review their work
- Note their perspective
- Consider their expertise

## Notes

- Always verify information from multiple sources
- Be transparent about source quality
- Update research as new information emerges
- Respect copyright and fair use
- Maintain objectivity in analysis

---

**Remember:** Good research is about quality, not quantity. Better to have 3 excellent sources than 10 mediocre ones.
